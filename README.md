# Android NoteBook 简易笔记本

## V 1.0.0
```aidl
com.menghuidream.notebook
targetSdkVersion 30
minSdkVersion 24
Gradle 7.0.2
Gradle Plugin 7.0
```

### 1. 项目简介
简易的笔记软件，使用Java开发，实现了添加笔记标题、内容，修改笔记标题、内容，删除笔记，搜索笔记等功能。

### 2. 注意
需在暗黑模式下才能正常显示，否则会出现看不见字体图标的情况。
